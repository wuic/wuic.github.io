Welcome to WUIC

Written in Java, the project focuses on best practices automation around statics. Its aims to guarantee high performance when your pages are loaded by the client on the web, keeping high productivity during your developments.



 Features

  • CSS & JS minification/aggregation with various processors
  • Various caching solutions & strategies
  • Image merging and sprites computation for both CSS and JS languages
  • Access to statics anywhere : classpath, disk, HTTP(S), FTP(S), SSH, Cloud
  • Processes resources referenced with @import in CSS files
  • JSP tag for better scalibility but easy to use with pure DHTML
  • Switch debug and production mode easily

More features coming soon »

 Why WUIC ?

  • Modular API to be integrated in various infrastructures and to keep scalibility
  • On the fly solution to automate and to be configurable as much as possible
  • Processes resources but also takes care about where to pick and to serve them
  • Fully documented and supported by many developers
  • Young has but already proved its strength in production
  • Very slim with minimal dependencies
  • Available on the maven central repository
  • Open source project under MIT license

 Quick start

You should start easily by taking a look at our different samples. You can see a sample showing how WUIC aggregates images, computes sprites and minifies Javascript for you ! Just move to the sample's directory and run it with : 

mvn jetty:run
